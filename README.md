# docker-ansible
A Docker Image, which contains Ansible, curl, wget, unzip and ssh.

The image tag refers to the Ansible version that is installed.

| Application | Version |
|-------------|---------|
| Ansible | 2.2.0.0 |
| curl | 7.43.0 |
| wget | 1.16.1 |
| unzip | 6.00 |
| ssh | 1:6.9p1-2ubuntu0.2 |
